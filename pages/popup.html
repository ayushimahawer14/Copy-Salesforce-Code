Description:

Easily copy the code by clicking the "Copy" button.

Instructions:

  1. Open any Apex Class, Apex Trigger, Visualforce Page, Visualforce Component and Formula Field.
  2. Click on the "Copy" button which is present along with other Salesforce standard buttons.
  3. An alert "Code Copied!" will be shown once code is copied to clipboard.
  4. Use the copied code.
